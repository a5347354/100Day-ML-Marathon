# Logistic Regression Model 邏輯斯回歸模型
## 介紹
屬於二元分類模型，可計算出A類別以及B類別的機率
## 優點
- 資料不需線性可分
- 可獲得A類別和B類別的機率
- 實務上執行速度快

## 缺點
- 分類不夠漂亮（線的切法不夠漂亮）
